# Phrase GraphQL API

Phrase is a localization platform for software translation. The API covers project management, key and translation management, file formats, locale handling, translation memory, machine translation, QA checks, and workflow automation.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Phrase REST API
**Documentation:** https://developers.phrase.com/api/
